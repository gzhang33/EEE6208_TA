# 数字与模拟电子学学习指南

[View English Version](README.md)

一个专为EEE6208数字与模拟电子学课程设计的综合性交互式学习指南，支持双语切换、交互式测验和掌握度跟踪。

## 🌟 功能特点

### 📚 **全面内容覆盖**
- **模拟电子学**: MOSFET工作状态、差分放大器、密勒效应、负反馈
- **数字电子学**: 逻辑设计、时序分析、物理设计、可测试性
- **交互式学习**: 可折叠章节、掌握度跟踪、自测测验

### 🎨 **现代用户体验**
- **双语支持**: 英文和中文之间的无缝切换
- **响应式设计**: 针对桌面、平板和移动设备优化
- **交互式导航**: 智能侧边栏，支持平滑滚动和活跃章节高亮
- **无障碍访问**: 符合WCAG标准，支持键盘导航和屏幕阅读器

### 🧮 **高级功能**
- **MathJax集成**: 高质量的LaTeX数学公式渲染
- **掌握度跟踪**: 可视化进度指示器，本地存储持久化
- **动态测验**: 自动生成自测问题，支持答案显示
- **打印优化**: 学习材料的清洁打印布局

### ⚡ **性能优化**
- **懒加载**: 高效的资源加载和渲染
- **防抖事件**: 平滑滚动和调整大小处理
- **CSS变量**: 一致的主题和易于定制
- **错误处理**: 强大的错误管理和优雅降级

## 🚀 快速开始

### 前置要求
- 现代网页浏览器（Chrome、Firefox、Safari、Edge）
- 无需额外软件安装

### 安装
1. 克隆仓库：
   ```bash
   git clone https://github.com/your-username/EEE6208_equations.git
   cd EEE6208_equations
   ```

2. 在浏览器中打开 `index.html`：
   ```bash
   # 使用Python（如果可用）
   python -m http.server 8000
   # 然后访问 http://localhost:8000/index.html
   
   # 或直接双击文件在浏览器中打开
   ```

### 使用指南

#### 导航
- **侧边栏**: 悬停（桌面）或点击（移动）左边缘打开导航
- **语言切换**: 点击右上角按钮切换语言
- **章节导航**: 使用大按钮在模拟和数字章节之间切换

#### 交互功能
- **可折叠内容**: 点击可展开章节显示额外信息
- **掌握度跟踪**: 使用表情按钮（😊🤔🤯）跟踪理解程度
- **自测评估**: 点击"生成测验"创建练习问题
- **回到顶部**: 向下滚动查看浮动导航按钮

#### 学习技巧
- 使用掌握度指示器跟踪每个主题的进度
- 定期生成测验测试理解程度
- 在语言之间切换以强化概念
- 打印章节进行离线学习（Ctrl+P）

## 🛠️ 技术实现

### 架构
```
EEE6208_equations/
├── index.html              # 主应用程序入口点
├── styles.css              # 包含变量和动画的综合CSS
├── app.js                  # 核心应用程序逻辑和状态管理
├── config.js               # MathJax配置和语言数据
├── content.js              # 双语内容定义
└── README.md              # 本文件
```

### 技术栈
- **前端**: 纯HTML5、CSS3、JavaScript（ES6+）
- **数学渲染**: MathJax 3.x用于LaTeX公式显示
- **样式**: CSS自定义属性（变量）用于主题
- **存储**: LocalStorage用于用户偏好和进度
- **性能**: 防抖事件、懒加载、优化动画

### 关键优化

#### 性能
- **资源预加载**: 关键脚本预加载以加快启动
- **防抖事件**: 平滑滚动和调整大小处理
- **高效DOM查询**: 缓存元素引用和安全选择器
- **MathJax优化**: 异步加载和回退渲染

#### 用户体验
- **渐进增强**: 核心功能无需JavaScript即可工作

- **无障碍访问**: ARIA标签、键盘导航、焦点管理
- **错误处理**: 对缺失元素或网络问题的优雅降级

#### 代码质量
- **模块化架构**: 关注点分离，函数组织清晰
- **状态管理**: 集中式应用程序状态和实用函数
- **错误边界**: 全面的错误处理和日志记录

## 🔧 自定义

### 添加新内容
1. 编辑 `content.js` 添加新章节
2. 在 `config.js` 中更新语言数据
3. 添加相应的掌握度控制和测验定义

### 样式更改
1. 修改 `styles.css` 中的CSS变量进行主题更改
2. 更新 `:root` 部分中的调色板
3. 自定义动画和过渡效果

### 功能扩展
1. 在 `app.js` 中添加新的交互组件
2. 在 `Utils` 对象中扩展实用函数
3. 在 `config.js` 中实现额外的语言支持
